ScheduleMaker Github

Here to help find the perfect schedule



How to use ScheduleMaker:

1. Click on the "Get Started" button.

2. Select how many classes you are taking.

3. Enter the name of the course and select how many sections are available.

3. Enter the section number, days of week, start time, and end time.

4. Select which schedule is perfect for you!
